# video2qr
***Video2QR is a web app that turns a video in a scannable QRcode for sharing.***

***How to use:*** 

+ ***Step 1 :*** *git clone https://github.com/geckoFooBar/video2qr.git*
+ ***Step 2 :*** *cd video2qr/frontend*
+ ***Step 3 :*** *npm install*
+ ***Step 4 :*** *ng serve -o*

**Follow the above steps to run the app.**
